🟢 네이버 블로그
네이버 로그인 방식: Playwright 브라우저 자동화로 직접 로그인합니다.
서버 .env 설정: NAVER_ID, NAVER_PW, NAVER_BLOG_ID
주의: 2단계 인증이 활성화된 경우 첫 로그인 후 신뢰 기기로 등록해두세요.
🟠 티스토리 OAuth 인증
1. 티스토리 앱 등록에서 앱을 만드세요
2. 서버 .env에 TISTORY_CLIENT_ID, TISTORY_CLIENT_SECRET 입력
3. 아래 버튼으로 OAuth 인증 후 발급된 토큰을 TISTORY_ACCESS_TOKEN에 저장
🔵 블로그스팟 (Google Blogger)
1. Google Cloud Console에서 OAuth 클라이언트 생성
2. Blogger API v3 활성화, Redirect URI: http://localhost:3000/oauth/blogger/callback
3. 서버 .env에 BLOGGER_CLIENT_ID, BLOGGER_CLIENT_SECRET 입력
4. 아래 버튼으로 인증 후 BLOGGER_REFRESH_TOKEN, BLOGGER_BLOG_ID 저장
